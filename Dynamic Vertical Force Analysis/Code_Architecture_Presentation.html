track_geometry.py — Overview
Converts cumulative distance [m] → local road geometry state at any point on the circuit
What It Does
Stores the Silesia Ring club circuit as 11 sequential TrackSegment objects
Each segment defines: turn radius, length, inclination, banking, ISO roughness class
Called 41,991 times — once per simulation timestep
Sign Convention

🛣 silesia_road_profile.py
silesia_road_profile.py
Generates the road surface displacement profile z_road(x) for one full lap
Two-Component Profile
① Stochastic: ISO 8608 PSD random profile — captures general roughness class
② Deterministic: Versine bumps at known kerb locations
ISO 8608 Power Spectral Density
Gd(n) = Gd(n₀) × (n / n₀)⁻ʷ n = spatial frequency [cycles/m] n₀ = 0.1 cycles/m (reference) w = 2.0 (waviness exponent)
Class B: 4×10⁻⁶ m³/cycle — smooth asphalt
Class C: 1.6×10⁻⁵ m³/cycle — infield tarmac
IFFT Synthesis Method
1. freqs = rfftfreq(n_pts, d=dx) 2. Gd = Gd_n0 × (freqs/n0)^(-w) 3. A = sqrt(Gd × df) 4. φ = random [0, 2π] 5. X = A × exp(iφ) 6. z = IFFT(X) → spatial profile
Versine Kerb Bumps
z_bump(x) = (h/2) × [1 - cos(π·x / L_half)]

🔧 vertical_dynamics.py
vertical_dynamics.py
2-DOF quarter-car ODE — computes dynamic tyre contact force Fz(t) from road input
Sprung Mass
ms = 80 kg
Body + half chassis
ks = 28,000 N/m
c = 1,500 N·s/m
RockShox Luft Air Spring
Unsprung Mass
mu = 12.5 kg
Wheel + hub + upright
kt = 220,000 N/m
Tyre vertical stiffness
z_road(t) — Road Input
↓ Fz_dynamic(t) = kt·(z_road − zu) + Fz_static
Equations of Motion
Sprung mass EOM: ms·z̈s = -c·(żs-żu) - ks·(zs-zu) Unsprung mass EOM: mu·z̈u = +c·(żs-żu) + ks·(zs-zu) - kt·(zu-z_road) Dynamic tyre force: Fz_dyn = kt·(z_road - zu) + (ms+mu)·g
Natural Frequencies (Validation)
fn_sprung = (1/2π)·√(ks/ms) = 2.98 Hz ✓ (1-4 Hz typical) fn_unsprung = (1/2π)·√((ks+kt)/mu) = 22.4 Hz ✓ ζ (damping) = c/(2·√(ks·ms)) = 0.28 ✓ (race suspension)

Adaptive stepping: large steps on smooth straights, tiny steps through kerbs — 10× faster than fixed-step loop
⚡ motor_torque_model.py
Physics-Based Motor Torque
Replaces constant 72 N·m with dynamic demand from rolling resistance, aero, and inertia
Energy Demand Model
Total resistive force: F_aero = 0.5 · ρ · Cd · A_frontal · v² F_roll = m · g · Crr F_accel = m · ax T_wheel = (F_aero + F_roll + F_accel) · r_tire T_motor = T_wheel / (eta_dt · gear_ratio)
Coasting Logic
If demand is negative (braking) or v ~ 0, T_motor = 0
Simulation now coasts 39% of the time, fixing unrealistic constant input bias

🚗 Vehicle_Dynamics_V2.py
PAC89 Magic Formula
The kremple() method — Pacejka 1989 semi-empirical tyre model
Lateral Force Formula
Fy = D·sin[C·arctan(B·α − E·(B·α − arctan(B·α)))] α = slip angle [rad] B = BCD / (C·D) stiffness factor C ≈ 1.3 shape factor (lateral) D = μ·Fz peak force capacity E = curvature factor BCD = cornering stiffness [N/rad]
Longitudinal: Discrete Scan
Scans slip ratio from −30% to +30% in small increments to find torque balance
Avoids singularity at zero slip (analytical inverse fails there)
Load Transfer Physics
(a) Pitch transfer from ax: ΔFz_pitch = m·ax·cg_h / wb ra_ΔFz_pitch = ΔFz_pitch × (l_f / wb) (b) Roll transfer from φ and Fy: M_roll = m_spr·g·del_h·sin(φ) + Fy·del_h ΔFz_roll = M_roll / (s/2) (c) Combined per corner: ra_Fz_out = Fz_static + ΔFz_pitch/2 + ΔFz_roll/2 ra_Fz_inn = Fz_static + ΔFz_pitch/2 − ΔFz_roll/2
Open Differential Bug Fix (V1→V2)
if turn_radius < 0: # Left turn inner_wheel = 'left' else: # Right turn inner_wheel = 'right' # Inner = full torque, Outer = zero

🎙 main_simulation.py
Dynamic Fz Injection
Critical coupling — how quarter-car result replaces static V2 vertical force
The Problem
Vehicle_Dynamics_V2 computes Fz from pitch + roll only — no road roughness
QuarterCar computes total dynamic Fz — no inner/outer split
The Solution: Ratio Scaling
# V2 gives the load DISTRIBUTION # QuarterCar gives the MAGNITUDE Fz_static_total = ra_Fz_out + ra_Fz_inn dyn_ratio_out = Fz_dyn_outer / (Fz_static_total / 2) dyn_ratio_inn = Fz_dyn_inner / (Fz_static_total / 2) Fz_out_final = ra_Fz_out × dyn_ratio_out Fz_inn_final = ra_Fz_inn × dyn_ratio_inn
Inner wheel Fz_dyn = mirror model: Fz_static − 0.6×(Fz_outer − Fz_static)

📈 Key Physics
Key Physics — Slip & Roll
Derivations for quantities not measured directly in the drive cycle
5.1 — Longitudinal Slip from ax
Drive cycle has no wheel speed sensor → slip must be estimated from ax
lon_slip = clip( (ax / 0.3g) × 30%, ±30% ) 0.3g = peak longitudinal capability (200 kg vehicle) At 0.3g → slip = 30% (PAC89 max useful slip) ax > 0 → positive slip (acceleration) ax < 0 → negative slip (braking)
5.3 — Inner/Outer Wheel ID
if turn_radius < 0: # Left turn inner = 'left' # lower-speed wheel else: # Right turn inner = 'right' # Inner → full motor torque (open diff) # Outer → zero drive torque # KEY BUG FIXED in V1 → V2
5.2 — Roll Angle from ay
Roll angle not measured directly — derived from lateral acceleration via suspension stiffness
φ = ay × roll_gradient [rad] roll_gradient = (m_spr × del_h) / K_roll_total K_roll_total = ks×(s_f/2)² + ks×(s_r/2)² Result for this vehicle: roll_gradient ≈ 0.273 deg/(m/s²) ≈ 2.7 deg/g
Lateral Acceleration Estimate
In corners (|R| < 9000 m and > 5 m): ay = v² / |turn_radius| On straights: ay = 0 Roll angle clamped to ±5°
